## Wave 1 Tasks

### T-01: Surat Jalan Modal Form

**File:** `client/src/pages/feed/SuratJalanModal.tsx`

**Task:**
- Create MUI Dialog component for Surat Jalan form
- Fields: tanggal, nomorSJ, plasmaId, feedProductId, jumlahZak, supplier
- Use React Hook Form + Zod validation
- Submit to `/api/feed/surat-jalan` (POST)
- On success: close modal, invalidate feedStock query, show success toast

**Acceptance:**
- [ ] Dialog opens from Feed Stock page "Tambah Surat Jalan" button
- [ ] All required fields validated (Zod schema)
- [ ] Submit creates feed movement and updates stock
- [ ] Error handling shows validation messages

---

### T-02: Feed Stock Page with DataGrid

**File:** `client/src/pages/feed/FeedStock.tsx`

**Task:**
- Create page with MUI DataGrid
- Columns: Plasma, Feed Type, Total Zak, Total Kg, Status
- Fetch from `/api/feed/stock` (GET)
- Add "Tambah Surat Jalan" button opening modal
- Low stock rows highlighted (isLow: true → warning color)

**Acceptance:**
- [ ] DataGrid loads stock data from API
- [ ] Columns sortable and filterable
- [ ] Low stock items show warning indicator
- [ ] "Tambah Surat Jalan" button opens modal

---

### T-03: Low Stock Alerts Panel

**File:** `client/src/components/feed/LowStockAlert.tsx`

**Task:**
- Create Alert component showing low stock items
- Filter: isLow = true from feed stock data
- Color coding: warning (yellow) for < 50 zak, error (red) for < 20 zak

**Acceptance:**
- [ ] Panel shows on Feed Stock page
- [ ] Items sorted by severity (red first)
- [ ] Click item scrolls to that row in DataGrid
