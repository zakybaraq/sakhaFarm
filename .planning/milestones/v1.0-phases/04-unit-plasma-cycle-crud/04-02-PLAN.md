Create server/src/modules/plasma/plasma.service.ts with 5 exported async functions:

**createPlasma(input: { unitId: number, name: string, farmerName?: string, address?: string, phone?: string, capacity?: number }, tenantId: number, userId: string)**
- Verify unit belongs to tenant: `db.select({ id: units.id }).from(units).where(and(eq(units.id, input.unitId), eq(units.tenantId, tenantId), isNull(units.deletedAt))).limit(1)`
- If not found → throw PlasmaNotInTenantUnitError
- Insert: `db.insert(plasmas).values({ ...input })`
- Fire-and-forget audit: try { db.insert(auditLogs).values({ userId, action: 'create', resource: 'plasma', resourceId: String(newId), newValue: JSON.stringify(input) }) } catch {}
- Return created plasma

**listPlasmas(tenantId: number, unitId?: number)**
- Base query: `db.select({ ...plasmas fields, unitName: units.name }).from(plasmas).leftJoin(units, eq(plasmas.unitId, units.id)).where(and(eq(units.tenantId, tenantId), isNull(plasmas.deletedAt)))`
- If unitId provided: add `eq(plasmas.unitId, unitId)` to where clause
- Order by: `desc(plasmas.createdAt)`
- Return array

**getPlasma(id: number, tenantId: number)**
- Query: `db.select({ ...plasmas fields, unitName: units.name }).from(plasmas).leftJoin(units, eq(plasmas.unitId, units.id)).where(and(eq(plasmas.id, id), eq(units.tenantId, tenantId), isNull(plasmas.deletedAt))).limit(1)`
- If result.length === 0 → throw PlasmaNotFoundError
- Return result[0]

**updatePlasma(id: number, input: { name?: string, farmerName?: string, address?: string, phone?: string, capacity?: number }, tenantId: number, userId: string)**
- Get existing: same query as getPlasma
- If not found → throw PlasmaNotFoundError
- Update: `db.update(plasmas).set(input).where(and(eq(plasmas.id, id), eq(plasmas.unitId, existing.unitId)))`
- Fire-and-forget audit with oldValue/newValue

**softDeletePlasma(id: number, tenantId: number, userId: string)**
- Get existing plasma (same as getPlasma)
- Check active cycles: `db.select({ count: count() }).from(cycles).where(and(eq(cycles.plasmaId, id), eq(cycles.status, 'active')))`
- If count > 0 → throw PlasmaHasActiveCyclesError
- Update: `db.update(plasmas).set({ deletedAt: new Date() }).where(eq(plasmas.id, id))`
- Fire-and-forget audit with action: 'delete'

Imports needed: { eq, and, isNull, desc, count } from 'drizzle-orm', { plasmas } from '../../db/schema/plasmas', { units } from '../../db/schema/units', { cycles } from '../../db/schema/cycles', { auditLogs } from '../../db/schema/audit_logs', { db } from '../../config/database'
