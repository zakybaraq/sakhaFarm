### README.md Template
```markdown
# Sakha Farm Management System

Poultry plasma farm management with Indonesian poultry farming domain.

## Tech Stack

- **Runtime**: Bun
- **Backend**: Elysia.js + Drizzle ORM
- **Frontend**: React 18 + Vite + MUI
- **Database**: MySQL
- **Testing**: Vitest

## Quick Start

\`\`\`bash
# Install dependencies
bun install

# Start services
docker-compose up -d

# Run development
bun run dev      # server
cd client && bun run dev  # client
\`\`\`

## Features

- Multi-tenancy (Unit → Plasma → Cycle → Recording)
- Feed inventory management
- Daily recording with FCR/IP calculation
- RBAC with dynamic permissions
```

### SETUP.md Template
```markdown
# Setup Guide

## Prerequisites

- Docker + Docker Compose
- Bun 1.0+

## Environment Variables

Create `server/.env`:

\`\`\`
DATABASE_URL=mysql://...
JWT_SECRET=min-32-chars-secret-key-here...
REDIS_URL=redis://...
\`\`\`

## Running

1. Start MySQL + Redis:
   \`\`\`bash
   docker-compose up -d
   \`\`\`

2. Install deps:
   \`\`\`bash
   bun install
   cd client && bun install
   \`\`\`

3. Run dev:
   \`\`\`bash
   bun run dev  # server :3000
   cd client && bun run dev  # client :5173
   \`\`\`
```
