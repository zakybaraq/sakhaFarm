       - Import lucia from server/src/auth.ts
       - Import { verifyRequestOrigin } from "lucia"
       - Create derive block that:
         a. CSRF check: if method !== GET, verify Origin header matches Host header using verifyRequestOrigin(originHeader, [hostHeader])
         b. Parse Cookie header manually (not Elysia Cookie API to prevent type coercion): `const cookieHeader = context.request.headers.get("Cookie") ?? ""`
         c. Extract sessionId: `lucia.readSessionCookie(cookieHeader)`
         d. If no sessionId → return { user: null, session: null }

         f. If session.fresh → set refreshed cookie via lucia.createSessionCookie(session.id)
         g. If !session → set blank cookie via lucia.createBlankSessionCookie()
         h. Return { user, session }
       - Export as Elysia plugin: `export const sessionMiddleware = new Elysia({ name: 'session-middleware' }).derive(...)`
    2. Wire into server/src/index.ts: `app.use(sessionMiddleware)` — apply globally
  </action>

    - Expired session sets blank cookie
    - { user, session } available on all request contexts
