## Key Decisions from CONTEXT.md (LOCKED - DO NOT REVISIT)

### D-01: Recording Validation Rules
- Future dates → Reject with 400 error
- Completed/Failed cycles → 400 error
- Day age: Auto-calculated from cycle.chickInDate + recording.date

### D-02: Auto-calculation Metrics (returned in responses)
- Cumulative Mortality (%) = (Total dead + Total culled) / Initial population × 100
- Deplesi (%) = (Initial population - Remaining population) / Initial population × 100
- SR (Survival Rate) % = (Remaining population / Initial population) × 100

### D-03: Standard Comparison Display
- Deviation in grams = Actual BW - Standard BW
- Deviation in % = ((Actual - Standard) / Standard) × 100
- Standard lookup: By DOC type + day age

### D-06: Schema Usage (already exists)
- daily_recordings table: cycle_id, recording_date, day_age, dead, culled, remaining_population, body_weight_g, feed_consumed_kg, notes
- standards table: doc_type, day_age, standard_bw_g, standard_fcr, daily_gain_g, cum_feed_intake_g
</context>
