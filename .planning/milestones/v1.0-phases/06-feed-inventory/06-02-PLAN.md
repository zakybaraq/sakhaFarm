Interface RecordConsumptionInput:
- plasmaId: number
- feedProductId: number
- recordingId: number (optional - links to daily_recording)
- consumptionKg: number
- consumptionZak: number (optional, computed from kg if not provided)
- notes: string (optional)

Function recordFeedConsumption(input, tenantId, userId):
1. Get current stock using getStockForPlasmaFeed
2. Calculate new_out = current.totalOutKg + input.consumptionKg
3. Calculate closing = opening + totalIn - new_out
4. If closing < 0, throw NegativeStockError(currentStock, requested)
5. Wrap in transaction:
   - Insert feed_movement (type='out', referenceType='recording', referenceId=recordingId)
   - Update feed_stock (increment totalOutKg by consumptionKg)
6. Audit log the consumption

Helper: getStockForPlasmaFeed(plasmaId, feedProductId, tenantId)
- Returns: { openingStockKg, totalInKg, totalOutKg, closingStockKg, closingStockZak }
- closingStockKg = opening + totalIn - totalOut
- closingStockZak = closingStockKg / zakKgConversion
</action>
