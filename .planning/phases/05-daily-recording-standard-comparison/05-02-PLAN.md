## Key Decisions from CONTEXT.md (LOCKED)

### D-04: Bulk Recording Format
- CSV columns: date, dead, culled, remaining_population, body_weight_g, feed_consumed_kg, notes
- Date format: YYYY-MM-DD
- Validation: All required columns, date format, non-negative numbers, cycle exists + active
- Error handling: Row-by-row, skip on error, continue
- Response: { success: true, created: N, errors: [{ row: 1, error: "..." }] }
- Limits: UTF-8 encoding, max 1000 rows

### D-05: IP Calculation Formula (at cycle completion)
- IP = (SR% × 0.4) + (BW_achievement% × 0.3) + (FCR_achievement% × 0.3)
- Components:
  - SR% = (Final population / Initial population) × 100
  - BW_achievement% = Actual BW / Standard BW × 100
  - FCR_achievement% = Standard FCR / Actual FCR × 100 (inverted)
- Calculate at cycle completion (POST /api/cycles/:id/complete)
</context>
