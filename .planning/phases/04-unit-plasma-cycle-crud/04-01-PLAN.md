3. `DuplicateUnitCodeError extends Error` — constructor(code: string), message: `Unit code "${code}" already exists`, name: 'DuplicateUnitCodeError'

Create server/src/modules/unit/unit.service.ts with 5 exported async functions:

**createUnit(input: { name: string, code: string, location?: string }, tenantId: number, userId: string)**
- Check for duplicate code: `db.select().from(units).where(and(eq(units.code, input.code), eq(units.tenantId, tenantId), isNull(units.deletedAt)))`
- If exists → throw DuplicateUnitCodeError
- Insert: `db.insert(units).values({ ...input, tenantId })`
- Fire-and-forget audit: try { db.insert(auditLogs).values({ userId, action: 'create', resource: 'unit', resourceId: String(newId), newValue: JSON.stringify(input) }) } catch {}
- Return created unit

**listUnits(tenantId: number)**
- `db.select().from(units).where(and(eq(units.tenantId, tenantId), isNull(units.deletedAt))).orderBy(desc(units.createdAt))`
- Return array

**getUnit(id: number, tenantId: number)**
- `db.select().from(units).where(and(eq(units.id, id), eq(units.tenantId, tenantId), isNull(units.deletedAt))).limit(1)`
- If result.length === 0 → throw UnitNotFoundError
- Return result[0]

**updateUnit(id: number, input: { name?: string, code?: string, location?: string }, tenantId: number, userId: string)**
- Get existing: same query as getUnit
- If not found → throw UnitNotFoundError
- If input.code provided → check duplicate excluding self: `and(eq(units.code, input.code), ne(units.id, id), eq(units.tenantId, tenantId), isNull(units.deletedAt))`
- If duplicate → throw DuplicateUnitCodeError
- Update: `db.update(units).set(input).where(and(eq(units.id, id), eq(units.tenantId, tenantId)))`
- Fire-and-forget audit with oldValue/newValue

**softDeleteUnit(id: number, tenantId: number, userId: string)**
- Get existing unit (same as getUnit)
- Check active plasmas: `db.select({ count: count() }).from(plasmas).where(and(eq(plasmas.unitId, id), isNull(plasmas.deletedAt)))`
- If count > 0 → throw UnitHasActivePlasmasError
- Update: `db.update(units).set({ deletedAt: new Date() }).where(and(eq(units.id, id), eq(units.tenantId, tenantId)))`
- Fire-and-forget audit with action: 'delete'

Imports needed: { mysqlTable, int, varchar, timestamp, index } from 'drizzle-orm/mysql-core', { eq, and, isNull, desc, ne, count } from 'drizzle-orm', { sql } from 'drizzle-orm', { units } from '../../db/schema/units', { plasmas } from '../../db/schema/plasmas', { auditLogs } from '../../db/schema/audit_logs', { db } from '../../config/database'

Every exported function MUST have JSDoc with @param and @returns.
