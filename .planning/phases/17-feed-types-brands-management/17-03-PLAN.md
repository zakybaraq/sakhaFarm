# Plan 03: Frontend Pages for Feed Types, Brands, and Products

## Objective
Create full CRUD frontend pages following Units/Plasmas pattern.

## Tasks

### Task 1: Feed Types Page
- Create `client/src/pages/feed/FeedTypes.tsx` — ResponsiveTable with columns: Name, Code, Status
- Create modal component for create/edit with fields: code, name
- Wire to API: listFeedTypes, create/update/toggle/delete feed-types
- Follow exact Units/Plasmas pattern: list → modal → confirm delete

### Task 2: Feed Brands Page
- Create `client/src/pages/feed/FeedBrands.tsx` — ResponsiveTable with columns: Name, Code, Phone, Status
- Create modal component for create/edit with fields: code, name, phone
- Wire to API: listFeedBrands, create/update/toggle/delete feed-brands

### Task 3: Feed Products Page
- Create `client/src/pages/feed/FeedProducts.tsx` — ResponsiveTable with columns: Code, Name, Type, Brand, Zak/Kg Conversion, Status
- Create modal component with fields: code, name, type (Autocomplete from feed_types), brand (Autocomplete from feed_brands), zakKgConversion
- Wire to API: listFeedProducts (updated), create/update feed products

### Task 4: API Client Updates
- Update `client/src/api/feed.ts`:
  - Add FeedType interface and CRUD functions (listFeedTypes, createFeedType, updateFeedType, toggleFeedType, deleteFeedType)
  - Add FeedBrand interface and CRUD functions
  - Update FeedProduct interface to include typeId, brandId, typeName, brandName
  - Update listFeedProducts, createFeedProduct, updateFeedProduct signatures

### Task 5: Routing & Sidebar
- Add routes in `client/src/App.tsx`:
  - `/feed/types` → FeedTypes
  - `/feed/brands` → FeedBrands
  - `/feed/products` → FeedProducts
- Add sidebar entries under "Feed" section (minimal, Phase 20 handles reorganization)

### Task 6: Update SuratJalanModal
- Update feed product dropdown to use updated API response (now includes type/brand info)

## Verification
- `cd client && npx tsc --noEmit` passes with zero errors
- All pages load and display data
- Create/edit/delete operations work end-to-end
