<objective>
Fix the frontend to read `isActive` from the API response and send `{ isActive: 0/1 }` on toggle instead of the defunct `{ isDeleted }`.

Purpose: The `Unit` and `Plasma` TypeScript interfaces currently declare `isDeleted: boolean`, which does not exist on the DB record. The toggle mutations send `{ isDeleted: !unit.isDeleted }` which the backend ignores (or rejects). This plan replaces `isDeleted` with `isActive: number` in both interfaces and fixes the toggle mutations and Switch `checked` props in both pages.

Output: Four modified files — two API modules and two page components. No new files created.
</objective>
