## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F5F5F5 | Page background, surfaces |
| Secondary (30%) | #FFFFFF | Cards, sidebar, navbar, modal surfaces |
| Accent (10%) | #2E7D32 | Primary buttons, active menu items, links, chart lines |
| Destructive | #D32F2F | Error states, delete actions |

**Accent reserved for:**
- Primary button backgrounds (contained variant)
- Active sidebar menu item backgrounds
- Link text color
- Performance chart actual data lines (#2E7D32)
- KPI trend up indicators

**Never use accent for:** DataGrid headers (use #f8fafc), form input borders (use default gray)

## API Patterns

| Pattern | Implementation |
|---------|--------------|
| Data fetching | TanStack Query useQuery from `client/src/api/feed.ts`, `client/src/api/reporting.ts` |
| Mutations | TanStack Query useMutation + onSuccess: queryClient.invalidateQueries(['feedStock']) |
| Form handling | React Hook Form + Zod schema (matching server-side validation) |
| Pagination | DataGrid built-in, default 25 rows |
| Export | xlsx library: Workbook → Worksheet → cell → writeFile |

## Summary

This UI-SPEC establishes the visual and interaction contracts for Phase 9 — Inventory & Reporting UI. Key decisions from CONTEXT.md are captured:

1. **Surat Jalan**: Modal popup for quick entry (D-01, D-02, D-03)
2. **Feed Stock**: DataGrid with filters (D-04, D-05, D-06)
3. **Performance Charts**: Line chart for BW deviation, Area chart for FCR (D-07, D-08, D-09)
4. **Export**: Excel (xlsx) format (D-10, D-11, D-12)
5. **Audit Log**: DataGrid with filters (D-13, D-14, D-15)
6. **RBAC Manager**: Single page with tabs (D-16, D-17, D-18)

Typography, colors, and spacing sourced from existing Phase 8 theme in `client/src/theme/index.ts`. All component inventory specified for executor clarity.
